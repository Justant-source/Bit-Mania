docker --version
docker compose version

# 3. Windows 재시작 후 자동 복구 확인

## Phase 1: API 키 발급 (예상 소요: 30분)

### Step 1-1. Bybit Testnet 계정 생성 + API 키

```

4. 우측 상단 프로필 → API → Create New Key
   - System-generated API Keys 선택

   - IP 제한: 없음 (테스트넷이므로)
   - 출금 권한: ❌ 절대 체크하지 않음
5. API Key와 Secret 복사
```

   [패널 2: 일별 수익률]

   [패널 3: 펀딩비 수취 히스토리]
   쿼리: SELECT collected_at, payment FROM funding_payments ORDER BY collected_at

   [패널 4: 전략별 수익 기여도]

   - 펀딩비 양수 → 포지션 진입 → 8시간 후 펀딩비 수취 → 로그 확인

5. 24시간 무중단 가동
   - 최소 3일 연속 가동 후 메모리 누수, 로그 크기 확인
