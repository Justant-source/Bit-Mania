## Test A — 3년 Walk-Forward 검증

### 목적

단기 과적합(overfitting) 가능성을 배제하고 실제 운용 가능성을 검증하기 위해,
2023~2026년 3년간 데이터를 학습 180일 / 검증 90일 단위로 슬라이딩하여 10개 Window에서 Out-of-Sample 성능을 집계.

### 데이터 기간 및 설정

| 항목 | 값 |
|------|-----|
| 전체 기간 | 2023-04-01 ~ 2026-03-31 |
| 학습 윈도우 (train) | 180일 |
| 검증 윈도우 (test) | 90일 |
| 총 윈도우 수 | 10개 |
| 데이터 출처 | `walk_forward_results` 테이블 |

### 결과

| 전략 | Aggregate Sharpe | Total Return | MDD | Consistency Ratio | 판정 |
|------|:---:|:---:|:---:|:---:|:---:|
| **funding_arb** | **2.434** | **+1.54%** | **0.38%** | **0.800** | **통과** |
| grid_trading | -0.581 | -14.89% | 35.82% | 0.000 | 불합격 |
| combined | -0.152 | -3.36% | 23.51% | 0.400 | 불합격 |

> **ALERT**: grid_trading 및 combined 포트폴리오의 Sharpe < 1.5 기준 미달.

### 분석

**funding_arb**
- 10개 윈도우 중 8개(consistency_ratio 0.800)에서 양의 Sharpe 달성.
- MDD 0.38%는 극히 낮은 수치로, 델타 뉴트럴 구조의 리스크 격리 효과가 실증됨.
- 3년이라는 긴 기간에 걸쳐 Sharpe 2.434를 기록한 것은 전략의 구조적 수익원(funding rate 수취)이 시장 레짐 변화에도 안정적임을 의미.

**grid_trading**
- consistency_ratio 0.000: 10개 윈도우 모두에서 음의 성과. 단 한 번도 양의 결과를 내지 못함.
- MDD 35.82%는 초기자본 대비 치명적인 손실 위험을 내포.
- 이는 BTC가 대형 추세 장세(급등/급락)로 전환될 때 그리드가 오히려 역추세 포지션을 누적하는 구조적 취약점을 재확인.

**combined (FA + Grid 혼합)**
- grid_trading의 부진이 funding_arb의 수익을 잠식하여 combined 전체 성과를 크게 훼손.
- consistency_ratio 0.400에 불과: grid 비중을 0으로 낮추는 것이 포트폴리오 안정성에 결정적으로 유리함.

## Test B — Funding Arb 변형 비교

### 목적

현재 funding_arb 전략의 청산(exit) 로직을 변형했을 때 성과 차이가 있는지 검증.
구체적으로 기존 exit 조건 대비 더 느슨한(tolerant) exit 조건을 적용한 변형을 비교.

### 데이터 기간 및 설정

| 항목 | 값 |
|------|-----|
| 검증 기간 | 2023-04-01 ~ 2026-03-31 |
| 데이터 출처 | `strategy_variant_results` WHERE test_name='test_b_funding_arb_variants' |

### 결과

| 변형 | Sharpe | Return | MDD | 거래수 | Win Rate |
|------|:---:|:---:|:---:|:---:|:---:|
| baseline | 0.3303 | +2.636% | 0.6648% | 103 | 43.689% |
| tolerant_exit | 0.3303 | +2.636% | 0.6648% | 103 | 43.689% |

### 분석

두 변형이 완전히 동일한 결과를 기록했다. 이는 다음 두 가지 해석이 가능하다:

1. **Tolerant exit 조건이 발동되지 않음**: 테스트 기간(3년) 중 baseline exit 기준이 항상 먼저 충족되어, tolerant 조건이 실질적으로 작동한 사례가 없었을 가능성이 높다.
2. **Tolerant exit의 추가 성과 기여 없음**: 현재 funding_arb의 exit 로직이 이미 충분히 최적화되어 있어, 조건을 완화해도 추가적인 수익 기회가 존재하지 않는다.

**결론**: 현재 baseline exit 로직을 유지하는 것이 적절하다. Tolerant exit 조건은 미래 시장 환경(예: 극단적 고변동성 구간)에서 유효성이 달라질 수 있으므로, 별도의 스트레스 테스트를 통해 재검증 필요.

**참고**: Test A의 Walk-Forward aggregate Sharpe 2.434와 Test B의 Sharpe 0.3303의 괴리는 측정 방식의 차이(Walk-Forward Out-of-Sample 집계 vs. 전 기간 단일 백테스트)에서 비롯된다. 두 결과는 상충하지 않으며, Walk-Forward 기준이 보다 실전에 가까운 지표임.

## Test C — DCA 필터 변형 비교

### 목적

adaptive-dca 전략의 진입 필터 및 포지션 누적 방식을 다양하게 변형하여,
현재 baseline 대비 개선 가능한 구조를 탐색.

### 데이터 기간 및 설정

| 항목 | 값 |
|------|-----|
| 검증 기간 | 2023-04-01 ~ 2026-03-31 |
| 데이터 출처 | `strategy_variant_results` WHERE test_name='test_c_dca_variants' |

### 결과

| 변형 | Sharpe | Return | MDD | Win Rate | 설명 |
|------|:---:|:---:|:---:|:---:|------|
| **graduated** | **0.2974** | **+13.89%** | **38.50%** | **40.76%** | **단계적 매수 (최우수)** |
| dual_filter | 0.2046 | +4.87% | 40.65% | 41.29% | Fear&Greed + RSI 이중 필터 |
| ema50_filter | 0.1026 | -5.79% | 43.85% | 40.88% | EMA50 추세 필터 추가 |
| baseline | -0.0069 | -20.43% | 59.26% | 40.16% | 현재 운영 방식 |

### 분석

**graduated 방식의 우수성**
- baseline 대비 return +34.32%p 개선, MDD는 오히려 59.26% → 38.50%로 대폭 축소.
- 단계적 매수(graduated DCA)는 하락 시 평균 단가를 낮추되, 추가 매수 규모를 단계별로 점진적으로 늘리는 방식으로 공격적인 물타기의 리스크를 완화.
- Win rate는 거의 동일(40%대)하지만, 승리 시 수익 규모가 더 크고 손실 시 노출이 낮은 비대칭 구조가 Sharpe 개선의 핵심.

**dual_filter**
- Fear&Greed 지수와 RSI 이중 조건으로 진입을 제한하여 baseline 대비 return +25.3%p, MDD 축소.
- graduated에는 미치지 못하나, 필터 자체의 유효성은 확인됨.

**ema50_filter**
- EMA50 방향성 필터 추가 시 오히려 수익률 악화 (return -5.79%).
- DCA는 본질적으로 역추세(counter-trend) 전략이므로, 추세 필터를 추가하면 핵심 진입 기회를 차단하는 역효과.

**baseline**
- 현재 운영 중인 adaptive-dca의 성과가 가장 열위(Sharpe -0.0069, return -20.43%).
- 3년 기간 중 BTC 하락 구간에서 무분별한 추가 매수로 MDD 59.26%까지 확대되는 위험 내포.

**권고사항**: adaptive-dca를 graduated 방식으로 업그레이드하는 것을 강력 권고.
업그레이드 전 반드시 orchestrator.yaml의 DCA 관련 파라미터(`dca_level_multiplier`, `max_dca_levels`)를 graduated 방식에 맞게 조정 필요.

## Test D — 그리드 대체 전략 비교

### 목적

grid_trading의 일관된 부진을 확인한 이후, 동일한 ranging(횡보) 레짐 환경에서
대체 전략이 유의미한 성과를 낼 수 있는지 탐색.

### 데이터 기간 및 설정

| 항목 | 값 |
|------|-----|
| 검증 기간 | 2023-04-01 ~ 2026-03-31 |
| 데이터 출처 | `strategy_variant_results` WHERE test_name='test_d_grid_replacement' |

### 결과

| 전략 | Sharpe | Return | MDD | Win Rate | 설명 |
|------|:---:|:---:|:---:|:---:|------|
| basis_spread | -0.4214 | -47.31% | 63.28% | 60.59% | 현물-선물 베이시스 스프레드 |
| vol_selling | -1.2523 | -30.67% | 31.38% | 61.09% | 변동성 매도 |
| mean_reversion | -2.0693 | -29.32% | 32.04% | 43.59% | 평균 회귀 |
| grid_baseline | -3.0945 | -87.77% | 88.05% | 45.74% | 기존 그리드 (비교 기준) |

### 분석

**전반적 부진**
- 4가지 대체 전략 모두 음수 Sharpe를 기록. 그리드의 대체 전략으로 ranging 레짐에서 구조적으로 유리한 방법을 찾는 것 자체가 어려움을 시사.

**basis_spread (상대적 최우수)**
- Sharpe -0.4214로 가장 낮은 음수 Sharpe이나, return -47.31%로 절대 손실은 가장 큼.
- Win rate 60.59%임에도 손실 폭이 크다는 것은 손실 시 규모가 승리 시 대비 현저히 크다는 의미 (Payoff Ratio 불리).

**vol_selling**
- Win rate 61.09%로 승률은 높으나 Sharpe -1.2523.
- 옵션 없이 선물만으로 변동성 매도를 구현할 때 발생하는 테일 리스크(tail risk)를 제어하지 못한 결과로 해석.

**mean_reversion**
- BTC 3년 기간 중 강한 추세 구간(2023~2024 상승, 2025 하락)에서 역추세 포지션 과다 누적으로 대규모 손실.

**grid_baseline**
- MDD 88.05%: 사실상 초기자본 전액 소멸 수준. Test A의 35.82%보다 훨씬 악화된 것은 테스트 조건 차이(레버리지, 포지션 크기 등)를 반영.

**결론**: ranging 레짐에서 그리드 및 모든 대체 전략의 성과가 일관되게 부진하므로,
ranging 레짐 판정 시 cash(현금 보유) 비중을 높이고 funding_arb에 집중하는 현재의 orchestrator.yaml 설정이 최적 선택임이 재확인됨.

**Ranging / Volatile 레짐의 견고성**

- **결론**: 가중치 파인튜닝보다 FA 전략 자체의 실행 품질(슬리피지, 체결 속도)이 더 중요한 성과 결정 요인.

**Trending Up 레짐**
- DCA 비중 증가 시 Sharpe 개선(0.7282 → 0.8035)되나, 개선 폭이 크지 않음(표준편차 0.029).
- graduated DCA 업그레이드 후 이 민감도 분석을 재실행하면 기울기가 더 가팔라질 것으로 예상.

**Trending Down 레짐**
- Cash 100% (FA 0%)이 가장 안전하나, FA 소량 유지 시(0.10~0.20) 추가 수익 가능.
- 단, trending_down 레짐에서 펀딩비는 음수(negative funding)로 전환될 수 있으므로,
  FA 비중 확대 결정 전 실시간 펀딩비 모니터링 조건 추가를 강력 권고.
- 현재 orchestrator.yaml의 trending_down FA 비중 설정값이 0.10~0.15 범위 이하인지 확인 요망.
