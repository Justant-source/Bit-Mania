### 필터 분석

```
총 TF 진입 신호:         3,719회
필터 차단:               713회 (19.2%)
차단 중 실제 손실이었던 건: 522회 (정확도 73.2%)
차단 중 실제 수익이었던 건: 191회
→ 필터 유의미 ✅ (정확도 73.2% >= 50% 기준)
```

**수익률 개선**:
- 연수익: -0.14% → +0.57% (+0.71%p)
- Sharpe: -0.020 → +0.180 (+0.200)
- MDD: 7.91% → 6.79% (-1.12%p)

> 펀딩비 필터는 유의미하게 작동하나, 필터 후에도 여전히 FA 단독보다 낮은 성과.

## 최종 결론: 추세추종 WF 실패 시 대안 경로

### 권장 대안 (우선순위 순)

#### 대안 1: FA + 수익 재투자 30% ← **즉시 적용 가능**

```yaml
# Phase 4에서 적용 가능한 설정
reinvest:
  enabled: true
  ratio: 0.30          # FA 수익의 30%를 BTC 현물 매수
  max_spot_pct: 20.0   # 포트폴리오 대비 현물 최대 20%
  
expected_improvement:
  portfolio_return: +59.24% → +78.63% (6년)
  annualized: 8.07% → 약 11%
  risk: 원금 리스크 없음 (수익분만 사용)
```

#### 대안 2: FA 자본 비중 50% → 70% 확대

```yaml
fa_capital_ratio: 0.70   # 현재 0.50
expected:
  annual_return: 8.07% × (0.70/0.50) ≈ 11.3%
  mdd: 0.25% × 1.4 ≈ 0.35%  (여전히 극도로 안전)
```

#### 대안 3: 멀티 심볼 FA (ETHUSDT, SOLUSDT 추가)

```yaml
symbols: [BTCUSDT, ETHUSDT, SOLUSDT]
expected:
  trade_frequency: 3x 증가 (연 ~100→300회)
  annual_return: 각 심볼별 독립 수익 누적
  correlation: 낮음 (각 심볼 펀딩비 독립)
```
