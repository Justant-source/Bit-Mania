| 실제 (현물+선물+슬리피지) | 0.0950% | 0.1900% | **$1.52** |
| **차이** | — | **+0.08%p** | **+$0.64/거래** |

### 원인 ② min_funding_rate 불일치


### 백테스터 조건으로도 수익인 이유

백테스터 조건으로는 **수익**이 납니다 ($+1.7944).
Production 코드의 파라미터 수정이 필요합니다.

### 즉시 수정 필요 항목

```python
# services/strategies/funding-arb/strategy.py
# 현재:
MIN_FUNDING_RATE = 0.00005  # 0.005%/8h — 백테스터 조건과 다름

# 수정:
MIN_FUNDING_RATE = 0.0001   # 0.01%/8h — 백테스터 조건과 동일
```

```yaml
# config/strategies/funding-arb.yaml
# 현재:
entry:
  min_funding_rate_annualized: 15.0  # 코드에서 읽히지 않음 (min_funding_rate 키가 없음)

# 수정: top-level에 명시적으로 추가
min_funding_rate: 0.0001  # 0.01%/8h = 연 10.95%, 백테스터 검증값
```

FA 전략은 **시장 환경에 강하게 의존**합니다. 백테스트 CAGR +34.87%는 6년 평균이며,

- 백테스터 조건: 10회 거래 / 1년, 각 거래 평균 166h 보유
