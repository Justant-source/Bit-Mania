# 그리드 트레이딩 전략

## 개요

현재 가격을 기준으로 위아래에 매수/매도 주문 그리드를 배치하여 횡보장에서 지속적으로 수익을 추구하는 전략입니다. 가격이 그리드를 오르내릴 때마다 소규모 차익을 실현합니다.

## 수익 구조

```
수익 = (체결된 그리드 쌍 수) x (그리드 간격 x 수량) - 수수료
```

- 횡보장에서 최적 성과
- 추세장에서는 한 방향으로만 체결되어 비효율적
- 변동성 급증 시 자동 비활성화

## 그리드 유형

### 기하급수 (Geometric) — 기본값
- 각 레벨 간격이 동일한 비율 (예: 0.5%)
- 가격이 높을수록 간격이 넓어짐
- 로그 정규분포를 따르는 가격에 적합

### 등차 (Arithmetic)
- 각 레벨 간격이 동일한 금액
- 좁은 가격 범위에서 유효
- 소규모 가격대 자산에 적합

## 그리드 생성

```
기하급수 그리드:
  상위 레벨: mid_price x (1 + spacing%)^i  (i = 1..N)
  하위 레벨: mid_price / (1 + spacing%)^i  (i = 1..N)

등차 그리드:
  상위 레벨: mid_price + step x i  (i = 1..N)
  하위 레벨: mid_price - step x i  (i = 1..N)
  step = mid_price x spacing%
```

### 기본 설정

| 파라미터 | 기본값 | 설명 |
|----------|--------|------|
| 유형 | geometric | 기하급수 간격 |
| 상위 레벨 수 | 10 | 매도 주문 수 |
| 하위 레벨 수 | 10 | 매수 주문 수 |
| 간격 | 0.5% | 레벨 간 비율 |
| 최소 간격 | 0.2% | 먼지 주문 방지 |
| 최대 간격 | 2.0% | 과도한 간격 방지 |
| 레벨당 주문 크기 | $200 | USD 기준 |

## 활성화 조건

| 조건 | 임계값 | 설명 |
|------|--------|------|
| 시장 레짐 | ranging | 횡보장에서만 활성화 |
| 레짐 신뢰도 | > 0.6 | 최소 60% 확신 |
| BB Width | < 0.06 | 볼린저밴드 폭 제한 |
| 24시간 거래량 | > $50M | 유동성 확보 |

## 그리드 리밸런싱

- **주기**: 60분마다 중심 가격 재계산
- **트리거**: 가격이 전체 그리드 범위의 30% 이상 이동 시
- **프로세스**:
  1. 기존 미체결 주문 전량 취소
  2. 새로운 중심 가격 계산
  3. 새 그리드 주문 배치

## 안전 장치

| 장치 | 임계값 | 동작 |
|------|--------|------|
| 최대 미체결 주문 | 40개 | 추가 주문 차단 |
| 최대 동시 그리드 | 2개 | 추가 그리드 생성 차단 |
| 비상 손실 | -$500 | 전 그리드 취소 |
| 스프레드 확대 | > 0.15% | 그리드 일시 중지 |
| 최소 체결 간격 | 5초 | 연속 체결 방지 |
| 체결가 편차 | > 0.5% | 이상 거래 감지 |
| 미체결 타임아웃 | 120분 | 주문 교체 |

## 리스크 관리

```yaml
risk:
  max_portfolio_allocation_pct: 30.0
  max_leverage: 2
  max_drawdown_pct: 4.0
  daily_loss_limit_usd: 300
  profit_target_per_cycle_usd: 500
```

- **최대 포트폴리오 비중**: 30%
- **최대 레버리지**: 2배
- **최대 낙폭**: 4% 초과 시 자동 종료
- **일일 손실 한도**: $300
- **사이클당 이익 목표**: $500 도달 시 그리드 리셋

## 브레이크아웃 대응

가격이 그리드 범위를 벗어나면:
1. `auto_deactivate_on_breakout: true` 시 자동 비활성화
2. 1% 버퍼 초과 시 전 주문 취소
3. 레짐 재평가 후 재활성화 결정

## 트레일링 그리드 (선택 기능)

```yaml
trailing_enabled: false
trailing_step_pct: 0.3
```

- 활성화 시 추세 방향으로 그리드 이동
- 상승 추세: 그리드 전체를 위로 0.3%씩 이동
- 하락 추세: 그리드 전체를 아래로 이동

## 백테스트 시 주의사항

- 횡보장 데이터에서 과적합 주의
- 슬리피지와 수수료를 반드시 포함
- 실제 체결률 (fill rate) 고려 필요
- 브레이크아웃 구간의 손실 확인 필수

## 최적 환경

| 환경 | 적합도 |
|------|--------|
| 횡보장 (BB Width < 0.04) | 최적 |
| 약한 추세 (ADX 15-20) | 양호 |
| 강한 추세 (ADX > 30) | 부적합 |
| 고변동 (ATR > 2x 평균) | 부적합 |
